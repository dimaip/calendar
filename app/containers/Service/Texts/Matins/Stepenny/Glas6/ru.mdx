#### Степенна, антифон 1, глас 6

**Н** К небесам возвожу очи мои — к Тебе, о Слово. / Смилуйся же надо мной, // дабы я жил для Тебя.

О Слово, помилуй нас, уничиже́нных, // соделай нас полезными для Тебя сосудами!

**[`Слава, и ныне.`]** Святой Дух — источник спасения всех людей. / Если кого удостоит Он дыханием Своим, / того тотчас возносит от дел земных: // окрыляет, укрепляет, устремляет ввысь.
