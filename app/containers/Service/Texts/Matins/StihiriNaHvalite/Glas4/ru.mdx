#### Стихира на «Хвалите», воскресная, глас 4

**Н** О всесильный Господи, / Крест и Смерть претерпевший / и воскресший из мёртвых, // мы прославляем Твоё Воскресение!

#### Стихира на «Хвалите», воскресная, глас 4

**Н** Твоим Крестом, о Христос, / Ты освободил нас от древнего проклятия / и Своей Смертью низложил диавола, / поработившего наше естество, / а Восстанием Своим Ты всё исполнил радостью. / Посему мы восклицаем Тебе: // «Воскресший из мёртвых, Господи, слава Тебе!»

#### Стихира на «Хвалите», воскресная, глас 4

**Н** Отеческого лона не покидая, / Ты, единородное Слово Божие, / пришёл на землю по Своему человеколюбию / и, неизменным пребывая, стал Человеком. / Ты претерпел по пло́ти крест и смерть, / по божеству непричастный страданию. / Воскреснув же из мёртвых, / Ты дарова́л бессмертие человеческому роду, // ибо Ты один всесилен.

#### Стихира на «Хвалите», воскресная, глас 4

**Н** О Спаситель, Ты при́нял по пло́ти смерть, / дабы бессмертие добыть для нас, / и во гроб вселился Ты, / дабы освободить нас из ада, воскресив вместе с Собою. / Ты пострадал как Человек, но воскрес как Бог! / Посему мы восклицаем: / «О Господь, Податель Жизни, слава Тебе, // один столь человеколюбивый!»
