#### Антифон 13-й, глас 6

Собрание иудеев просило Пилата / распять Тебя, Господи! / Вины́ в Тебе не найдя, / они виновного Вара́вву освободили, / а Тебя, Праведного, осудили, / грех скверного убийства получив в удел. / Но не воздай им, Господи, // тщетное против Тебя замыслившим!

Кого страшится и пред Кем трепещет всё, / Кого всякий язык воспевает — / Христа, Божию Силу и Божию Премудрость, — / священники били по щека́м / и давали Ему желчь. / И всё претерпеть Ты изволил, / Своею Кровью желая спасти нас от беззаконий наших // как человеколюбивый.

**Слава. Богородичен:** О Богородица, ты через слово, что превыше всякого разума, / родила своего Создателя — // моли же Его о спасении душ наших!

**И ныне.** О Богородица, ты через слово, что превыше всякого разума, / родила своего Создателя — // моли же Его о спасении душ наших!

#### Антифон 14-й, глас 8

Господи, Ты взял Себе в спутники разбойника, / руки осквернившего кровью! / Сопричти́ и нас к нему // как благой и человеколюбивый.

Разбойник, малое слово вымолвив на кресте, / великую веру обрёл, / в один миг спасён был / и первым, врата отворив, вошёл в рай. // О Господь, приня́вший его покаяние, слава Тебе!

**Слава. Богородичен:** Радуйся, от ангела приня́вшая радость для мира; / радуйся, родившая своего Творца и Господа; // радуйся, удостоившаяся стать Матерью Божией!

**И ныне.** Радуйся, от ангела приня́вшая радость для мира; / радуйся, родившая своего Творца и Господа; // радуйся, удостоившаяся стать Матерью Божией!

#### Антифон 15-й, глас 6

Ныне То́т висит на древе, / Кто подвесил землю среди вод; / венцом из терния уве́нчан ангелов Царь; / в плащ багряный облачён Одевающий небо облаками; / пощёчины воспри́нял во Иордане Освободивший Адама; / гвоздя́ми пригвождён Жених Церкви; / пронзён копьём Сын Девы. / Поклоняемся страданиям Твоим, Христе! / Поклоняемся страданиям Твоим, Христе! / Поклоняемся страданиям Твоим, Христе! — // Яви нам и Твоё славное Воскресение!

Не как иудеи будем праздновать, / ведь Пасха наша — за́кланный за нас Христос. / Очистим же себя от всякой скверны / и в чистоте помолимся Ему: / «Восстань, Господи, спаси нас // по Своему человеколюбию!»

Крест Твой, Господи, / для народа Твоего есть жизнь и воскресение, / и, надеясь на него, / мы Тебя, распятого Бога нашего, воспеваем. — // Помилуй нас!

**Слава. Богородичен:** Видя Тебя, о Христос, вися́щим на кресте, / родившая Тебя взывала: / «Что́ есть тайна непостижимая, которую вижу я, о Сын мой? / Ка́к Ты, о Жизни Податель, умираешь, // плотью пригвождённый на древе?»

**И ныне.** Видя Тебя, о Христос, вися́щим на кресте, / родившая Тебя взывала: / «Что́ есть тайна непостижимая, которую вижу я, о Сын мой? / Ка́к Ты, о Жизни Податель, умираешь, // плотью пригвождённый на древе?»

#### Седален, глас 4

Ты выкупил нас из проклятия закона / драгоценной Кровью Своей. / Ко кресту пригвождённый / и копьём пронзённый, / Ты людям бессмертие излил. // О Спаситель наш, слава Тебе!
