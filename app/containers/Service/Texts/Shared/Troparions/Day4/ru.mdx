#### Тропарь четверга, св. апостолов, глас 3

Святые апостолы, / хода́тайствуйте пред милостивым Богом, / о дарова́нии ду́шам нашим // согрешений прощения!

#### Тропарь четверга, свт. Николая, глас 4

Мерило веры и образ кротости, / учителя воздержности / паства твоя познала в тебе, / увидев дела твои. / Ты обрёл в смирении величие, / в нищете — богатство, / посему, иерарх и отец наш Николай, / хода́тайствуй пред Христом Богом // о спасении душ наших!
