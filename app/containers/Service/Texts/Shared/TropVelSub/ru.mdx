#### Тропарь Великой Субботы, глас 2

Достопочтенный Иосиф, / сняв с креста непорочное тело Твоё, / плащаницей чистой с ароматами его обвил / и, погребая, // в гробнице новой положил.

#### Тропарь Великой Субботы, на «Слава», глас 2

Когда к Смерти сошёл Ты, бессмертная Жизнь, / тогда ад умертвил Ты сиянием божества. / Когда же Ты и уме́рших из преисподней возвёл, / все силы небесные воскликнули: // «О Жизни Податель, Христос Бог наш, слава Тебе!»

#### Тропарь Великой Субботы, на «И ныне», глас 2

Мироносицам жёнам / у гроба предстал ангел и воскликнул: / «Миро уме́ршим подобает, / Христос же явился // чуждым тления».
