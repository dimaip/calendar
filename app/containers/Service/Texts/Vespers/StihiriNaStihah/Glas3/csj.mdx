#### Стихира на стиховне, воскресная, глас 3

**Н** Стра́стию Твое́ю, Христе́, / омрачи́вый со́лнце / и све́том Твоего́ Воскресе́ния, / просвети́вый вся́ческая, // приими́ на́шу вече́рнюю песнь, Человеколю́бче.

#### Стихира на стиховне, воскресная, глас 3

**Стих** Госпо́дь воцари́ся, / в ле́поту облече́ся.

**Н** Живоприе́мное Твое́ воста́ние, Го́споди, / вселе́нную всю просвети́, / и Твое́ созда́ние истле́вшее призва́. / Те́мже кля́твы Ада́мовы изме́ншеся, вопие́м: // Всеси́льне Го́споди, сла́ва Тебе́.

#### Стихира на стиховне, воскресная, глас 3

**Стих** И́бо утверди́ вселе́нную, / я́же не подви́жется.

**Н** Бог сый неизме́нен, / пло́тию стражда́ измени́лся еси́, / Его́же тварь не терпя́щи ви́сяща зре́ти, / стра́хом прекланя́шеся, / и стеня́щи пое́т Твое́ долготерпе́ние: / соше́д же во ад, тридне́вен воскре́сл еси́, // жизнь да́руя мíрови, и ве́лию ми́лость.

#### Стихира на стиховне, воскресная, глас 3

**Стих** До́му Твоему́ подоба́ет святы́ня, / Го́споди, в долготу́ дний.

**Н** Да род наш от сме́рти, Христе́, изба́виши, / смерть претепе́л еси́: / и тридне́вен из ме́ртвых воскре́с, / с Собо́ю воскреси́л еси́, / и́же Тя Бо́га позна́вших: / и мiр просвети́л еси́. // Го́споди, сла́ва Тебе́.

#### Стихира на стиховне, воскресная, на Слава, и ныне, богородичен, глас 3

**Н** Без се́мене от Боже́ственнаго Ду́ха, / во́лею же О́тчею зачала́ еси́ Сы́на Бо́жия, / от Отца́ без ма́тере пре́жде век су́ща: / нас же ра́ди, из Тебе́ без отца́ бы́вша, / пло́тию родила́ еси́, / и Младе́нца млеко́м пита́ла еси́. / Те́мже не преста́й моли́ти, // изба́витися от бед душа́м на́шым.
