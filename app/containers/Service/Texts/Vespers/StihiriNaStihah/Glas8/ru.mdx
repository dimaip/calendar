#### Стихира «на стихах», воскресная, глас 8

**Н** Восшёл Ты на крест — о Иисус, / с Небес сошедший! / На смерть Ты пришёл, о бессмертная Жизнь, / к сидящим во тьме — Свет истинный, / к падшим — Восстание всех! // О *наше* Просвещение и Спаситель наш, слава Тебе!

#### Стихира «на стихах», воскресная, глас 8

**Стих** Господь — Царь. / Его одея́ние — слава.

**Н** Восславим Христа, воскресшего из мёртвых, / ибо, восприня́в душу и тело, / Он Своим страданием расторг *их связь*: / непорочная душа сошла во ад, обезоружив его, / а святое тело Избави́теля душ наших // в гробнице не увидело тления.

#### Стихира «на стихах», воскресная, на «Слава, и ныне», богородичен, глас 8

**Н** О не познавшая брака Дева, / неизреченно по пло́ти зача́вшая Бога, / Матерь Бога Вышнего, / прими моления чад своих, Всенепорочная, / всем подавая очищение от согрешений. / Ныне, услышав наши молитвы, // хода́тайствуй о спасении всех нас!
