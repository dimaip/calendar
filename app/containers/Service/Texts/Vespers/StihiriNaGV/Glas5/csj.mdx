#### Стихира на Господи воззвах, воскресная, глас 5

**Н** Честны́м Твои́м Кресто́м, Христе́, / диа́вола посрами́л еси́, / и Воскресе́нием Твои́м жало грехо́вное притупи́л еси́, / и спасл еси́ ны от врат сме́ртных: // сла́вим Тя, Единоро́дне.

#### Стихира на Господи воззвах, воскресная, глас 5

**Н** Воскресе́ние дая́й ро́ду челове́ческому, / я́ко овча́ на заколе́ние веде́ся, / устраши́шася сего́ кня́зи а́дстии, / и взя́шася врата́ плаче́вная. / Вни́де бо Царь сла́вы Христо́с, / глаго́ля су́щим во у́зах, изыди́те: // и су́щим во тьме, откры́йтеся.

#### Стихира на Господи воззвах, воскресная, глас 5

**Н** Ве́лие чу́до, / неви́димых Соде́тель, за человеколю́бие пло́тию пострада́в, / воскре́се Безсме́ртный. / Прииди́те оте́чествия язы́к, Тому́ поклони́мся: / благоутро́бием бо Его́ от пре́лести изба́вльшеся, // в Трие́х Ипоста́сех Еди́наго Бо́га пе́ти навыко́хом.

#### Стихира на Господи воззвах, воскресная, глас 5

**Н** Вече́рнее поклоне́ние / прино́сим Тебе́, Невече́рнему Све́ту, / на коне́ц веко́в, я́ко в зерца́ле пло́тию возсия́вшему мíрови, / и да́же до а́да низше́дшему, / и та́мо су́щую тьму разруши́вшему, / и свет Воскресе́ния язы́ком показа́вшему: // Светода́вче Го́споди, сла́ва Тебе́.

#### Стихира на Господи воззвах, воскресная, глас 5

**Н** Нача́льника спасе́ния на́шего, / Христа́ славосло́вим: / Тому́ бо из ме́ртвых воскре́сшу, / мiр от пре́лести спасе́н бысть. / Ра́дуется лик А́нгельский, / бе́гает де́монов пре́лесть, / Ада́м пады́й воста́, // диа́вол упраздни́ся.

#### Стихира на Господи воззвах, воскресная, глас 5

**Н** И́же от кустоди́и науче́ни быва́ху от беззако́нник, / покры́йте Христо́во Воста́ние, / и приими́те сре́бреники, и рцы́те / я́ко нам спя́щим, из гро́ба укра́ден бысть Ме́ртвый. / Кто ви́де, кто слы́ша, мертвеца́ укра́дена когда́, / па́че же пома́зана и на́га, / оста́вльша и во гро́бе погреба́льная Своя́? / Не прельща́йтеся иуде́е, / навы́кните рече́нием проро́ческим, и уразуме́йте, // я́ко Той есть вои́стинну Изба́витель мíра, и Всеси́льный.

#### Стихира на Господи воззвах, воскресная, глас 5

**Н** Го́споди, ад плени́вый, / и смерть попра́вый, Спа́се наш, / просвети́вый мiр Кресто́м честны́м, // поми́луй нас.

#### Стихира на Господи воззвах, воскресная, на Слава, и ныне, богородичен, глас 5

**Н** В Чермне́м мо́ри, / Неискусобра́чныя Неве́сты о́браз написа́ся иногда́: / та́мо Моисе́й, раздели́тель воды́: / зде же Гаврии́л, служи́тель чудесе́. / Тогда́ глубину́ ше́ствова немо́кренно Изра́иль; / ны́не же Христа́ роди́ безсе́менно Де́ва. / Мо́ре по проше́ствии Изра́илеве пребы́сть непрохо́дно; / Непоро́чная по Рождестве́ Эмману́илеве пребы́сть Нетле́нна. / Сый, и пре́жде Сый, / явле́йся, я́ко Челове́к, // Бо́же поми́луй нас.
