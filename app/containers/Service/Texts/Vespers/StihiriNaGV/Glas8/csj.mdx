#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Вече́рнюю песнь, / и слове́сную слу́жбу, / Тебе́, Христе́, прино́сим, / я́ко благоволи́л еси́ // поми́ловати нас Воскресе́нием.

#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Го́споди, Го́споди, / не отве́ржи нас от Твоего́ лица́, // но благоволи́ поми́ловати нас Воскресе́нием.

#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Ра́дуйся, Сио́не Святы́й, / ма́ти церкве́й, Бо́жие жили́ще, / Ты бо прия́л еси́ пе́рвый, // оставле́ние грехо́в, Воскресе́нием.

#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Е́же от Бо́га Отца́ Сло́во, / пре́жде век Ро́ждшееся, / в после́дняя же времена́, / То́ежде от Неискусобра́чныя вопло́щшееся во́лею, / распя́тие сме́ртное претерпе́: / и дре́вле умерщвле́ннаго челове́ка спасе́ // Свои́м Воскресе́нием.

#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Е́же из ме́ртвых Твое́ Воскресе́ние славосло́вим Христе́, / и́мже свободи́л еси́ Ада́мский род от а́дова мучи́тельства: / и дарова́л еси́ мíрови я́ко Бог жизнь ве́чную // и ве́лию ми́лость.

#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Сла́ва Тебе́, Христе́ Спа́се, / Сы́не Бо́жий Единоро́дный, / пригвозди́выйся на Кресте́, // и воскресы́й из гро́ба тридне́вен.

#### Стихира на Господи воззвах, воскресная, глас 8

**Н** Тебе́ сла́вим, Го́споди, / во́лею нас ра́ди Крест претерпе́вшаго, / и Тебе́ покланя́емcя, Всеси́льне Спа́се, / не отве́ржи нас от Лица́ Твоего́, / но услы́ши и спаси́ ны // Воскресе́нием Твои́м, Человеколю́бче.

#### Стихира на Господи воззвах, воскресная, на Слава, и ныне, богородичен, глас 8

**Н** Царь Небе́сный / за человеколю́бие на земли́ яви́ся, / и с челове́ки поживе́: / от Де́вы бо Чи́стыя плоть прие́мый, / и из Нея́ проше́дый с восприя́тием. / Еди́н есть Сын, сугу́б естество́м, / но не Ипоста́сию. / Те́мже соверше́нна Того́ Бо́га / и соверше́нна Челове́ка вои́стину пропове́дающе, / испове́дуем Христа́ Бо́га на́шего: / Его́же моли́, Ма́ти Безневе́стная, // поми́ловатися душа́м на́шим.
