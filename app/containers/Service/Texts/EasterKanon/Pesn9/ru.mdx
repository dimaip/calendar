**Припев** Величает душа моя / воскресшего на третий день из гроба // Христа, Жизни подателя!

**Ирмос**	Светись же, светись, новый Иерусалим, / ибо Слава Господня над тобою взошла. / Ныне торжествуй и веселись, Сион, / а ты, чистая Богородица, радуйся // восстанию Рождённого тобой!

**Припев** Христос — новая Пасха, / жертва живая, / Агнец Божий, // уносящий грехи мира!

Голос Твой, о Христос, — / божественный, любимый и сладчайший, / ибо Ты обещал непреложно быть с нами до скончания века. / Имея *это слово* как якорь надежды, // мы, верные, радуемся!

**Припев** Ангел воскликнул Благодатной: / чистая Дева, радуйся! / И снова скажу: радуйся! / Твой Сын восстал / в третий день из гроба / и воздвиг мёртвых – // люди, возвеселитесь!

О Христос — Пасха великая и святейшая, / Премудрость, Божие Слово и Сила! / Дару́й нам полнее к Тебе приобщаться // в невечерний день Царства Твоего!

**Катавасия** Светись же, светись, новый Иерусалим, / ибо Слава Господня над тобою взошла. / Ныне торжествуй и веселись, Сион, / а ты, чистая Богородица, радуйся // восстанию Рождённого тобой!

Христос воскрес из мёртвых, / Смертию смерть поправ // и тем, кто во гробах, Жизнь дарова́в. **(Трижды, поско́ру.)**
