**Ирмос** Отроков от печи́ Избавивший, / став Человеком, / страдает, словно смертный, / и в Его страдании / смертное облекается в красоту нетления. / Он один благословен и препрославлен — // Бог отцов наших.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Жёны богомудрые / с миром за Тобою устремились, / они Тебя искали со слеза́ми, словно смертного, / но как живому Богу, / радуясь, поклонились Тебе / и тайну Пасхи, о Христос, // Твоим ученикам возвестили.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Смерти празднуем мы умерщвление, / ада разрушение, / иной, вечной, Жизни начало / и, ликуя, воспеваем / это Совершившего, / ибо Он один благословен и препрославлен — // Бог отцов наших.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Воистину эта спасительная ночь — / священная, праздничная и светозарная, / она — светоносного дня воскресения предвестница / В эту ночь негасимый Свет // из гроба во плоти всем воссиял.

**Катавасия** Отроков от печи́ Избавивший, / став Человеком, / страдает, словно смертный, / и в Его страдании / смертное облекается в красоту нетления. / Он один благословен и препрославлен — // Бог отцов наших.

Христос воскрес **(или:** восстал**)** из мёртвых, / Смертию Смерть поправ // и тем, кто во гробах, жизнь дарова́в! **(Трижды, поско́ру.)**
