**Ирмос** Отроков от печи́ Избавивший, / став Человеком, / страдает как смертный, / и Своим страданием / облекает смертных в красоту нетления. / Он один благословен и препрославлен — // Бог отцов *наших*.

**Припев**

Жёны богомудрые / с миром за Тобою устремились. / Они Тебя как смертного искали со слезами, / но, радуясь, поклонились Тебе как живому Богу / и возвестили, что Ты — Пасха сокровенная, // о Христос, Твоим ученикам.

**Припев**

Мы празднуем смерти умерщвление, ада разрушение, / иной, вечной, Жизни начало / и, ликуя, воспеваем это Совершившего, / ибо Он один благословен и препрославлен — // Бог отцов *наших*.

**Припев**

Воистину эта спасительная ночь — / священная, праздничная и светозарная, / она — светоносного дня Воскресения предвестница. / В эту ночь негасимый Свет // из гроба во плоти всем воссиял.

**Катавасия** Отроков от печи́ Избавивший, / став Человеком, / страдает, словно смертный, / и Своим страданием / облекает смертных в красоту нетления. / Он один благословен и препрославлен — // Бог отцов *наших*.

Христос воскрес из мёртвых, / Смертию смерть поправ // и тем, кто во гробах, Жизнь дарова́в. **(Трижды, поско́ру.)**
