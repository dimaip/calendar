**Ирмос** На стражу, ожидая Бога, / пророк Авваку́м да встанет с нами и укажет / на светоносного ангела, громко восклицающего: / «В сей день — спасение миру, / ибо восстал Христос, // ведь Он всесилен».

**Припев**

Христос есть *первенец* мужеского пола, / разверзший девственную утробу. / Как снедь Он наречён Агнцем, / а непорочным — как чуждый скверны. / Он — наша Пасха, // и как Бог истинный назван совершенным.

**Припев**

Словно однолетний ягнёнок, / Христос за всех был за́клан по воле Своей: / Он — Пасха очищающая, / Он наш благословенный Венец! / И снова из гроба прекрасное // Солнце Правды нам воссияло.

**Припев**

Пра́отец Давид скакал и плясал / пред ковчегом, что был тенью *будущих благ.* / Мы же, святой Божий народ, / видя исполнение прообразов, / в божественном веселии радуемся, / ибо восстал Христос, // ведь Он всесилен.

**Катавасия** На стражу, ожидая Бога, / пророк Авваку́м да встанет с нами и укажет / на светоносного ангела, громко восклицающего: / «В сей день — спасение миру, / ибо восстал Христос, // ведь Он всесилен».

Христос воскрес из мёртвых, / Смертию смерть поправ // и тем, кто во гробах, Жизнь дарова́в. **(Трижды, поско́ру.)**
