### Третий антифон, глас 4, Пс 109:1-3

~~1~~ Слово Господа к Госуда́рю моему: «Воссядь одесну́ю Меня,

**Припев** Рождество Твоё, Христе Боже наш, / озарило мир светом познания, / ведь тогда звёздам служа́щие / звездою были научены / поклоняться Тебе, Солнцу Правды, / и познавать Тебя, Зарю свыше. — // Господи, слава Тебе!

~~2~~ пока не повергну врагов Твоих в подножие ног Твоих!»

**Припев** Рождество Твоё, Христе Боже наш, / озарило мир светом познания, / ведь тогда звёздам служа́щие / звездою были научены / поклоняться Тебе, Солнцу Правды, / и познавать Тебя, Зарю свыше. — // Господи, слава Тебе!

~~3~~ Ски́петр силы Твоей от Сиона пошлёт Господь: владычествуй посреди врагов Твоих!

**Припев** Рождество Твоё, Христе Боже наш, / озарило мир светом познания, / ведь тогда звёздам служа́щие / звездою были научены / поклоняться Тебе, Солнцу Правды, / и познавать Тебя, Зарю свыше. — // Господи, слава Тебе!

~~4~~ С Тобою власть в день силы Твоей во свете святынь.
